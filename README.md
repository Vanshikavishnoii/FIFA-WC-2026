# FIFA World Cup 2026 Dashboard

A modern football dashboard/web application built with **TypeScript, JavaScript, and Tailwind CSS**. This project displays FIFA World Cup 2026 related information in a clean and responsive interface.

## Features

* Responsive dashboard UI
* Built with TypeScript
* Tailwind CSS styling
* Fast and modern frontend setup
* Organized component structure
* Mobile-friendly design

## Tech Stack

* **TypeScript**
* **JavaScript**
* **Tailwind CSS**
* **Node.js**
* **Bun / npm**

## Project Structure

* `components/` – reusable UI components
* `public/` – static assets
* `verify-dashboard.png` – dashboard preview
* `verify-final.png` – final UI preview
* `tailwind.config.ts` – Tailwind configuration
* `tsconfig.json` – TypeScript configuration

## Installation

Clone the repository:

```bash
git clone https://github.com/Vanshikavishnoi/FIFA-WC-2026.git
```

Go to the project folder:

```bash
cd FIFA-WC-2026
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
